HTML Basics

The "frontend" of a webpage consists of:

  • HTML = HyperText Markup Language
    • content
  • CSS = Cascading Style Sheet
    • style
  • JS = Javascript
    • interactivity

Vocabulary

elements have the following syntax: <tag attribute="value">content</tag>

Block vs. Inline

block-level elements always starts on a new line and take up the full width available inline-elements do not start on a new line and only takes up as much width as necessary
